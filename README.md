# vscode不能自动补全结束标签？
文件-->首选项-->设置。在搜索设置中键入“tab”关键词，找到结果  "emmet.triggerExpansionOnTab": false。在右侧的用户设置中添加一个键值对------ "emmet.triggerExpansionOnTab": true ，覆盖默认设置，done。
